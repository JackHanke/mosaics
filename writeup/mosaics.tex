An $(n,m)$-\textit{mosaic} is a rectangular grid made up of tiles.

\begin{center}
\begin{tikzpicture}
% row1

\cellC{1}{0}{2}{1}
% row2
\cellA{0}{1}{1}{2}
\cellB{1}{1}{2}{2}
\end{tikzpicture}

we have that $t_{n,1}=t_{1,m}=0$, and $t_{2,2} = 1$. What else can be said?
